Introduction

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

HTML uses "markup" to annotate text, images, and other content for display in a Web browser. HTML markup includes special "elements".

Image Source: MDN

An HTML element is set off from other text in a document by "tags", which consist of the element name surrounded by < and >. The name of an element inside a tag is case insensitive. That is, it can be written in uppercase, lowercase, or a mixture.

Base elements

There are 6 base elements within the HTML code, which are the following:

  1. <html>
  2. <head>
    1. <meta>
    2. <link>
    3. <script>
  4. <body>

These tags form the basic layout of a simple webpage, in the order of appearance in the list above.

<html>
    <head>
        <meta charset="UTF-8">
        <link rel="stylesheet" href="example.css">
        <script src="example.js"></script>
    </head>
    <body>
        <!-- here comes your code -->
    </body>
</html>

This what a base of webpage's code looks like.

Semantics

Everything that follows here comes obligatorily into the <body> element.

  1. <article>
  2. <aside>
  3. <figcaption>
  4. <figure>
  5. <footer>
  1. <header>
  2. <main>
  3. <mark>
  4. <nav>
  5. <section>
  6. <time>

Let's see an example of the amalgam of these tags.

<main>
  <article>
    <header>
      <nav><!-- example navigation menu--></nav>
    </header> 
    <section>
      <p><mark>Example</mark> text.</p>
    </section>
    <footer>
      <p>Example footer © 
      <time datetime="2021-06-15T21:13">2021
      </time>
      </p>
    </footer>
  </article>
</main>

<aside>
    <section>
      <p>Example text.</p>
    </section>
    <figure>
      <img src="example.png" alt="alternative text">
      <figcaption>Descriptive text about the figure.
      </figcaption>
    </figure>
</aside>

There is an order of appearance of the most important tags, as follows <main>,<aside>,<article>,<header>,<nav>,<section>,<footer>.

Forms

There are a many elements which occur in a <form> field, but let's just name a few that appear most frequently.

  1. <button>
  2. <fieldset>
  3. <form>
  4. <input>
  5. <legend>
  6. <textarea>

Now let's take a look at an example subscription form.

<form action="/subscribe" method="POST">
    <fieldset>
      <legend>Subscribe for more</legend>
      <input type="email" name="email">
      <button>Subscribe</button>
    </fieldset>
</form>
Lists

Lists make our life easier when it comes to ordering our tasks, just as in this case. We have ordered list, unordered list, these two have their own list-items and there is definition list for more advanced uses.

  • <dd>
  • <dl>
  • <dt>
  • <li>
  • <ol>
  • <ul>

Let's see some examples.

<ol> <!-- ordered list, with numbered items-->
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ol>

<ul> <!-- unordered list, with bulleted/dotted items-->
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<dl>
  <dt>Web</dt>
  <dd>The part of the Internet
  that contains websites and web pages</dd>
  <dt>HTML</dt>
  <dd>A markup language for creating web pages</dd>
  <dt>CSS</dt>
  <dd>A technology to make HTML look better</dd>
</dl>
Tables

The <table> HTML element represents tabular data — that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.

  • <caption>
  • <table>
  • <tbody>
  • <td>
  • <tfoot>
  • <th>
  • <thead>
  • <tr>

Let's see a pleasent example of a table.

<table>
  <caption>The Beatles</caption>
  <tr>
    <td>John Lennon</td>
    <td>Rhythm Guitar</td>
  </tr>
  <tr>
    <td>Paul McCartney</td>
    <td>Bass</td>
  </tr>
  <tr>
    <td>George Harrison</td>
    <td>Lead Guitar</td>
  </tr>
  <tr>
    <td>Ringo Starr</td>
    <td>Drums</td>
  </tr>
</table>
